Introductie over mij

Mijn naam is Shi Hua Liu, ik ben gestart met programmeren in 2020. ik ben een jonge programeur dat de laatste paar jaren heeft zitten focussen op HTML en CSS. Momenteel zou ik me meer willen uitbreiden in Javascript en CSS, omdat ik daar niet veel ervaring mee heb. Momenteel leren we de basis van React en de manier hoe we met React werken spreekt me heel erg aan.

Om meer ervaring te krijgen in andere velden wil ik graag stage lopen.

meer informatie over mezelf: Ik ben geboren in 2004 en ik ben opgegroeid in Hoorn. Mijn hobbies zijn: uitgaan met vrienden, muziek luisteren, games spelen, manga's lezen.
